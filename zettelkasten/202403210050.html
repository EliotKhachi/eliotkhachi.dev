Learning Blockchain

  1. Know the opportunity of blockchain [Read Write Own: Building the Next Era of the Internet]
  2. Know the Major Use Cases of Blockchain
  3. Know what kind of blockchain developer you want to be (Application Dev or Core / Infrastructure Dev).
  4. What programming languages to learn? What blockchain should I use? Solidity - primary language to build smart contracts with Ethereum (Ethereum has over 50% of the value in blockchains)
  5. Other tools to learn:
  • Hardhat framework- Helps you build and test smart contracts locally for development.
  • Remix- A workspace / online IDE to experiment with smart contracts on your browser.
  • EthersJS - Must-know Javascript library that lets you interact with smart contracts with built-in authentication using client-side private-keys.
  • Metamask - A browser extension that gives you a wallet that enables you to connect to blockchains. You need a wallet to connect to / sign any blockchain.
  • ReactJS
  • NextJS
  • NodeJS
  1. Step by Step Process
    • Go use a blockchain and gain some familiarity with the user experience. Install Metamask and use uniswap. Buy an NFT on OpenSea.
    • Do a lightweight tutorial project that covers smart contracts with solidity and how the front-end interacts with it.
    • Tackle a big project with no guidance. Add features little by little.
