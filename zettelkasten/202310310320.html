Roadmap for Switching Careers to Tech

  1. Learning my first language
  2. Learning Web Development (Full Stack Development Course covering JavaScript, React, MySQL, Docker, Web Security)
  3. Learning Data Science (Python, Machine Learning, etc.)
  4. Learning AWS and Building a Side Project
  5. Preparing for Code Interviews
    • Prepare Theory (~1.5 months)
      • Google / YouTube "The must know data structures and algorithms for coding interviews"
      • Watch System Design YouTube videos.
      • Create zettels. Dive deep. Note time complexity, memory complexity, learn the data structures' syntax for a language (i.e. Java), and note app architecture
    • Prepare Coding Questions (~2 months)
      • Do over 100 problems. Repeat some of the hard ones.
    • Prepare Behavioral Questions
      • Google "20 most common behavior questions asked in interviews for a software engineering role" and prepare answers for them.
  6. Apply to jobs while practicing coding problems
