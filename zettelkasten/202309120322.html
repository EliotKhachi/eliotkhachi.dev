Redis

Redis (Remote Dictionary Server) is an in-memory data store that can be used as a database, cache [What is Caching?], streaming engine, or message broker.

Amazon ElastiCache Redis supports...

  • Multi AZ with Auto-Failover
  • Read replicas
  • Data durability using AOF persistence
  • Backup and restore features
  • Suppors sets and sorted sets
